PHP Gas-Mileage Database

PGMDB is a minimalistic web application designed to be easy to install, and easy to use. Please read this guide before using the application to become familiar with how it works.

Getting Started

Create a vehicle file

Choose the "create" option from the database function drop-down menu. Click "Submit Query". Enter a new filename, and a password for adding records. Fill out the rest of the information and click "Submit Query".

Create refueling records

Now you may start adding refueling records using the "add record" option from the database function drop-down menu.

Viewing Records

There at least three different record views:

  • Detailed View
  • Print View
  • Plot View

Detailed View   This is the view that is normally used. It has three major section: records and derived statistics (those that aren't stored in the file), a summary (statistics highlights section), and a detailed statistics section with many interesting facts (min/avg/max/latest).

Print View   This the view that should be used to print out the records for archival. Only the important information is displayed to save paper.

Plot View   This view helps you see trends. It requires gnuplot to be installed and accessable by your web server.

How the Math Works

Here's the basic assumption: the amount of fuel that it takes to fill your tank is the amount of fuel that was spent since the last time the tank was filled. Therefor, if you fill each time, then the latest mile/gallon estimate is just the distance you travelled (difference of odometer readings) divided by the amount of fuel you purchased.
