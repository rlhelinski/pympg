My PHP Gas-Mileage (MPG) User Manual

This script is a monolithic (single-file) PHP script which manages refueling records. It is extended to also record mantenance records.

Getting Started

Create a vehicle file

Choose the "create" option from the database function drop-down menu. Click "Submit Query". Enter a new filename, and a password for adding records. Fill out the rest of the information and click "Submit Query".

Create refueling records

Now you may start adding refueling records using the "add record" option from the database function drop-down menu.
